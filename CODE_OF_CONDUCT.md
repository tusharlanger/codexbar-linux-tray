# Code of Conduct

This project adopts the **Contributor Covenant v2.1**.

Read the full text here: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be kind, be patient, assume good intent, focus on the technical problem. Behaviour that makes the project hostile or unproductive for others is not welcome.

## Reporting

If you experience or witness behaviour that violates this code, please email the maintainer at **tushar.langer@gmail.com**. Reports are handled privately and in good faith.
